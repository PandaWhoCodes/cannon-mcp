# Service: Frontend

## Getting Context

Use the `grounding-docs` MCP server for cross-service context:

- `get_architecture` — system-wide architecture
- `get_api_contracts` — how services communicate
- `get_doc("frontend", "CLAUDE.md")` — this service's detailed context
- `get_doc("backend", "CLAUDE.md")` — backend context (when working on API integration)
- `search_docs("keyword")` — find any doc mentioning a topic
